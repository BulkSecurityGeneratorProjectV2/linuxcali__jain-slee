Mobicents JOPR Plugin for Mobicents JAIN SLEE 2.x Server
========================================================

This is the JOPR plugin for providing management and monitoring 
for Mobicents JAIN SLEE 2.x Server under JBoss JOPR platform.

How to use
----------

Use this plugin with the latest version of RHQ Server (1.4.0+), 
which can be found at:

http://sourceforge.net/projects/rhq/files/



Find installation instruction here: 

http://community.jboss.org/wiki/Jopr-Installation
