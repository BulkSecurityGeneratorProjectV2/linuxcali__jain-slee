SIP11 Dialog Fork Test Application
---------------------------------------------------
This is a test application for the SIP11 RA. Its 
code should not be taken as example.

Requirements: 
---------------------------------------------------
1. SIPP in OS path environment
2. Mobicents JAIN SLEE 2.x with SIP11 RA deployed
3. JBOSS_HOME pointing to Mobicents JAIN SLEE 2.x 

Running:
---------------------------------------------------
1. install sip11 ra
2. install this example
3. start mobicents
4. run batch files in sipp directory (one at a time!)
