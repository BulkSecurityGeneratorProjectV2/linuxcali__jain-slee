Description:
This example is goolgle talk(xmpp) bot.

Requirements:
This example requires valid google account. User needs to provide credentials in sbb-jar.xml file.

Install/Uninstall:
To install/uninstall google talk example and its dependencies, use the ant script "build.xml":

- ant deploy-all, deploys xmpp ra and the example DU
- ant undeploy-all, undeploys xmpp ra and the example DU
